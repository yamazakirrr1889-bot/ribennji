# Ribennji Tasks

ブラウザだけで動く、シンプルなタスク管理 Web アプリです。

## できること

- タスクの追加、編集、削除
- ステータス管理: 未着手、進行中、確認、完了
- ボード、リスト、集中ビューの切り替え
- 検索、優先度フィルター、並び替え
- ドラッグアンドドロップでステータス変更
- localStorage への自動保存
- JSON のエクスポート、インポート

## 使い方

`index.html` をブラウザで開くだけで使えます。

## ファイル構成

- `index.html`: 画面構造
- `styles.css`: UI スタイル
- `app.js`: タスク管理ロジック
- `assets/ribennji-mark.svg`: アプリマーク
